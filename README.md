# Structural Identity Theory (SIT)

**Author / المؤلف:** Wasfi Ahmed Mohamed Al-Shehadeh / وصفي أحمد محمد الشحادة

**Registration No / رقم التسجيل:** F4000 : 62896

## Overview / نبذة

**EN:** This repository contains the mathematical proofs and computational models for the Structural Identity Theory (SIT). The theory presents black holes as maximal singularities of matter and proposes mechanisms (dual vaporization) explaining cosmic / galactic expansion through processes at event horizons.

**AR:** يحتوي هذا المستودع على البراهين الرياضية والنماذج الحسابية لنظرية الهوية الهيكلية. تثبت النظرية أن الثقوب السوداء تمثل أقصى تفرد للمادة وتقترح آليات (تبخير مزدوج) تفسر التوسع الكوني عبر عمليات عند أفق الحدث.

## Key Scientific Principles / المبادئ العلمية الرئيسية

- Maximum Singularity / التفرد الأقصى: Black holes are the maximum mass limit due to nuclear shell collapse into dense quarks.
- Zero Density Propagation / انتشار ذات كثافة صفرية: Light conceived as zero-density propagation; optical bending interpreted as visual effect.
- Dual Vaporization / التبخير المزدوج: Galactic expansion driven by fast electron vaporization and slow quark vaporization at horizons.
